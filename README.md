# 赵梓寓 | Ziyuzhao - Portfolio

个人作品集单页，采用深色模式（Dark Mode）与极简主义设计风格，全方位展示教育背景、荣誉奖项、核心工作实习经历以及数据分析相关的技术技能。

## 🌟 核心特色 (Features)

- **高级暗色视觉 (Premium Dark Mode)**: 使用深灰色至纯黑色的线性与径向渐变背景，配合微小噪点纹理，辅以沉稳的暗金色为强调色，体现“投资银行/战略咨询”的专业与质感。
- **动态光晕互动 (Interactive Cursor Glow)**: 结合 JavaScript 实现的鼠标跟随光晕效果（全局光晕以及卡片内独立悬浮光影），赋予页面灵动的现代科技感。
- **极简卡片化经历 (Card-based Layout)**: 将核心实习经历提炼并封装为交互卡片，运用 CSS 毛玻璃（Backdrop-filter）与悬停微浮雕效果，突出重点信息。
- **优雅字体排版 (Typography)**: 混用 `Playfair Display`（衬线体处理标题以彰显典雅）与 `Inter`（无衬线体处理正文以保证阅读体验）。
- **完全响应式 (Fully Responsive)**: 基于 CSS Grid 和 Flexbox 的流式布局，完美适配桌面与移动设备。

## 🚀 技术栈 (Tech Stack)

- **HTML5**: 清晰的语义化结构
- **CSS3**: 原生 CSS，包含复杂渐变、毛玻璃质感、CSS 变量、过渡动画
- **JavaScript (Vanilla)**: 零依赖的原生 JS 处理 DOM 和鼠标监听事件

## 📂 项目结构 (Project Structure)

```text
├── index.html     # 主页面 HTML 骨架与数据呈现
├── style.css      # 全局样式、动画特效与响应式适配
└── script.js      # 鼠标跟随发光互动逻辑
```
